hello, this is just an explanation for the studio part alone.

apis:
/studio/ - get will give all studios, post will create a studio

/studio/id - get will fetch that particular studio, patch can update the details of the studio

/studio/id/members- lists all members

/studio/id/add-member- will add a member

/studio/id/members/member_id - patch and delete will update or delete the member.
